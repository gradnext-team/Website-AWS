# gradnext Brand Guidelines

## ⚠️ IMPORTANT: Brand Name Usage

**The brand name must ALWAYS be written in lowercase: `gradnext`**

### ❌ INCORRECT
- gradnext
- gradnext
- Gradnext
- GRADnext

### ✅ CORRECT
- gradnext

## Where This Applies

This lowercase convention applies EVERYWHERE in the codebase:
- Email subjects and body text
- Calendar invite titles and descriptions
- UI text (buttons, labels, headings, tooltips)
- Legal pages (Terms, Privacy Policy, Refund Policy)
- Error messages and alerts
- Payment gateway display names
- Meta tags and page titles
- Comments and documentation
- API responses
- Log messages

## Email Addresses
All email addresses use lowercase domain:
- info@gradnext.co
- support@gradnext.in
- noreply@gradnext.co

## Social Media Handles
- linkedin.com/company/gradnext
- instagram.com/gradnext
- twitter.com/gradnext

## URLs
- https://gradnext.co
- https://app.gradnext.co

---

**Note to Developers/AI Agents:** When making any changes involving the brand name, always use lowercase `gradnext`. This is a strict brand guideline.
