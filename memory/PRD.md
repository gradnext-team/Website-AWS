- 2026-05-11: P0 CRITICAL — **Production Cloudflare edge-caching fix for mentor images**. Replaced unconditional `GZipMiddleware` with a `ConditionalGZipMiddleware` that skips compression on `/api/images/*`, `/api/files/*`, `/api/stream/*`. Reason: gzip-ing already-compressed JPEGs caused Cloudflare to mark them `cf-cache-status: DYNAMIC` (refused to edge-cache), making mentor images load in 2-6s per scroll. Post-fix: images flow with `Cache-Control: public, max-age=31536000, immutable` and NO `content-encoding: gzip` → Cloudflare edge-caches → ~50-100ms loads. JSON responses keep gzip (15 KB → 1.6 KB). File: `backend/server.py`.
